Upload CSV

CSV File Instructions:

The CSV file should contain the following columns:

  • original_text: The original sentence to be translated
  • translation_method: The method used for translation
  • model1_name: The name of the first translation model
  • model1_translation: The translation produced by the first model
  • model2_name: The name of the second translation model
  • model2_translation: The translation produced by the second model
  • model3_name: The name of the third translation model
  • model3_translation: The translation produced by the third model

Example:

original_text,translation_method,model1_name,model1_translation,model2_name,model2_translation,model3_name,model3_translation
"Hello, world!",neural_machine_translation,GPT-3,"Hola, mundo!",NLLB,"Bonjour, le monde!",Google Translate,"Ciao, mondo!"
"How are you?",statistical_machine_translation,GPT-3,"¿Cómo estás?",NLLB,"Comment allez-vous?",Google Translate,"Come stai?"
